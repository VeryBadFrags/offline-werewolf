    ⚙️ Game Settings


    Rules

    • 👥 3-10 Players
    • ⏱ 7 Minutes Rounds
    • 📱 1 Device per Player
    Setup:
    • All players must input the same 🎲 Game Code, 🔢 Round Number, and 👥 Total players count
    • Players must pick a unique 👤 Avatar
    • When the game starts, players must confirm that they have the same Phase Fingerprint (the game is void otherwise)
    Game phases:
    • During the Night Phase, players pick an action and share their Action Code with the others
    • Players must input the Action Code for the other players and end the turn
    • When the Day Phase starts, players must confirm that they have the same Phase Fingerprint (the game is void otherwise)
    • During the Day Phase, players share their knowledge and try to identify who the werewolves are
    • At the end of the timer, players must vote for a player to Exile
    • The game ends when a majority is reached - if there is no majority, the werewolves win
    Win conditions:
    • Werewolves and Cultists win if a Villager or a Cultist is exiled (the exiled Cultist also wins)
    • Villagers win if a 🐺 Werewolf is exiled
    • Everyone loses if the 🤡 Jester is exiled
