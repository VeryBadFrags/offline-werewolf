# Offline Werewolf

A server-free, fast-paced spin of the Werewolf formula.

* [Play the game](https://wolf.verybadfrags.com)

## Build

Minify and bundle the css and js into a standalone HTML page.

* Get [Node](https://nodejs.org/)

* Build
    ```shell
    make
    ```
