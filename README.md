A server-free, fast-paced version of the Werewolf / Mafia formula.

* [Play the game](https://wolf.verybadfrags.com)

## Build

Minify and bundle the css and js into a standalone HTML page.

* Install [Node](https://nodejs.org/)

* Build
    ```sh
    make
    ```
